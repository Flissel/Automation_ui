# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

## [1.0.0] - 2026-02-21

### Added
- Real-time desktop streaming via WebSocket (multi-monitor support)
- AI-powered screen analysis with Video Agent (Nemotron VL)
- Guardian Mode: automatic error correction for desktop actions
- Monitor Mode: real-time screen change detection
- Node-based workflow automation system (14 node types)
- OCR integration (Tesseract, EasyOCR, PaddleOCR)
- Remote desktop control (mouse, keyboard, scroll)
- LLM Intent Agent for natural language desktop commands
- MCP Server integration (32 tools for desktop automation)
- Docker Compose setup for containerized deployment
- Desktop streaming client with dual-monitor capture
